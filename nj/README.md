## 获取南京地区租房信息

### 各个文件作用
`crawl.py`:爬取租房信息   
`rent.csv`:房源信息文件  
`index.html`:显示地图  

### 实现功能
仅爬取58同城 个人房源信息  
存在爬取过程中断问题，且不准备深入研究  
我已经看不懂JS了...  
### 使用方法
1. 进入 amap-Python/nj
1. 运行`crawl.py`脚本
1. 运行web服务 python -m htto.server 8080
1. 打开浏览器 输入 http://localhost:8080
1. 选择工作地点
1. 将csv文件导入到地图中
#### JS逻辑
* 获取基本地图
* 添加标尺
* 添加输入提示查询 获取地址名称
* 正向地理编码 根据地址名称 获取坐标
* 根据坐标 添加公交到达圈
* 读取csv文件名
* 打开csv文件，按行存入数组中，截取地址存入新的数组
* 根据地址添加房源标记
